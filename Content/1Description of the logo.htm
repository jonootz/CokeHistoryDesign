Description of the logo

 

 

There are various versions that are currently being used, however the main components of the official logo are made of three elements.

Lettering

The absolute main component of the logo is the lettering. The base logo uses a script type font called "" which was a popular style in the late 1900's. This was the original logo except it is more refined now whereas originally it was handwritten.

Colors

The colors are red and white. The basic logo is the red lettering with a transparent background. But it is more commonly used inverted with the red for the background and white is used for the lettering depending on where and how it is used.

Wave

The "dynamic ribbon" is another element that usually is placed below the lettering. It is called many things such as the swirl, the spiral, the wave, the ribbon. The color of the wave matches the color of the lettering.

In addition to the main logo, Coca-Cola uses a few more designs to represent them and they are very important to their corporate identity.
